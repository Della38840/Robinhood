# Robinhood API Documentation

**Unofficial Documentation of Robinhood Trade's Private API**

---

## Project Description

Robinhood is a commission-free, online securities brokerage. This project documents the unofficial private API used by Robinhood's trading platform. Documentation fills blanks found in Robinhood's official docs.

- **Official Docs:** https://docs.robinhood.com/crypto/trading/
- **GitHub Discussions:** https://github.com/sanko/Robinhood/discussions

## API Overview

| Section | Description |
|---------|-------------|
| Authentication | Login, tokens, password recovery |
| Banking | Bank accounts & ACH transfers |
| Order | Placing, cancelling, listing orders |
| Options | Options related endpoints |
| Quote | Stock quotes |
| Fundamentals | Basic fundamental data |
| Instrument | Financial instruments reference |
| Watchlist | Watchlists management |
| Account | User and account information |
| Settings | Notifications and settings |
| Markets | Exchange information |
| Referrals | Account referrals |
| Statistics | Social/statistical endpoints |
| Tags | Categorizing endpoints |
